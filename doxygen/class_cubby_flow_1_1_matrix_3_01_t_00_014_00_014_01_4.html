Detailed Description

template<typename T>
class CubbyFlow::Matrix< T, 4, 4 >

4-D matrix class.

This class is a row-major 4-D matrix class, which means each element of the matrix is stored in order of (0,0), ... , (0,3), (1,0), ... , (3,3). Also, this 4-D matrix is specialized for geometric transformations.

Template Parameters
T- Type of the element.

Constructs a matrix with given initializer list list.

This constructor will build a matrix with given initializer list list such as

{1, 2, 4, 3},
{9, 3, 5, 1},
